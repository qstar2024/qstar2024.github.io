How to Play Dice Game

Welcome to the 3D Dice Rolling Game! Here's how to play:

Game Instructions:

  • Click the "Roll Dice" button to roll the dice.
  • The dice will tumble and settle, displaying their face values.

Dice Patterns:

  • 秀才/一秀 has one die with value of 4.
  • 举人/二举 has two dices with value of 4.
  • 进士/四进 four dices of the same value (e.g., four 6s).
  • 探花/三红 has three dices with value of 4.
  • 榜眼/对堂 all six dices showing different numbers. (a.k.a., 1,2,3,4,5,6)
  • 状元 has four dices with value of 4.
  • 特殊规则 you'll figure out as you play along!
